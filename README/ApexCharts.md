# Apex Charts

- API에서 받아온 데이터들을 시각화 하는 라이브러리(그래프 등)

> step1)

```
npm install --save react-apexcharts apexcharts
```

> step2)

```typescript
import ApexChart from "react-apexcharts";
```

> step3) set types

- line
- bar
- candle
- etc...

```typescript
<ApexChart type="line" options={{}} series={{}} />  --> 각각 attribute는 공식문서에서 참고
```
